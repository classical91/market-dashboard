Setup:
1. Create a bot via @BotFather and copy the token.
2. Add the bot to each channel as an admin.
3. Set TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_IDS (comma-separated channel IDs) in your Railway environment variables.
4. Redeploy — the next fresh report will auto-post to all channels.
